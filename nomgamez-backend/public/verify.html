🎰 NOM-GAMEZ — Provably Fair Verification

Enter the session details below to verify any game outcome independently.

How it works:

  • Server commits to SHA256(serverSeed) at session start
  • You provide clientSeed + nonce
  • Outcome = HMAC-SHA256(serverSeed, clientSeed:nonce:game)
  • After the round, server reveals serverSeed for verification
